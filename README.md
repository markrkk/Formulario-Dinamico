# Formulario Dinámico

Formulario de registro desarrollado con HTML, CSS y JavaScript.

## Descripción

Este proyecto permite al usuario completar un formulario de registro con nombre, email, contraseña, país y aceptación de términos.

El formulario incluye validaciones simples y muestra mensajes en pantalla sin usar alertas.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Captura de datos con FormData
- Uso de entries()
- Validación de nombre vacío
- Validación básica de email
- Validación de contraseña mínima
- Validación de checkbox de términos
- Uso de preventDefault()
- Evento submit
- Evento reset
- Mensajes dinámicos en pantalla
- Diseño responsive básico

## Objetivo del proyecto

Practicar:

- Formularios
- Eventos
- Validaciones
- FormData
- entries()
- Manipulación del DOM
- preventDefault()
- Mensajes dinámicos
